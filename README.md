# Vanilla Node REST API

> Full CRUD REST API using Node.js with no framework

[link](https://www.youtube.com/watch?v=_1xa8Bsho6A&t=3086s) to the Traversy Media video

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install
yarn install

# Run in develpment
npm run dev
yarn run dev

# Run in production
npm start
yarn start
```
